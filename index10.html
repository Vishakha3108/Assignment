Q.9 Create an orderd list of the most common text formatting tags in HTML. Within each list item,use an unordered list the specific usa cases and best practices for that tag.
Ans. Formatting elements were designed to display special types of text:
  1. b tag
    • The html b tag defines bold text,without any extra importance.
  2. strong tag
    • The html strong tag defines text with strong importance.The content inside is typically displayed in bold
  3. i tag
    • The html i tag element definnes a part of text in an alternate voice or mood.The content inside is typically displayed in italic.
  4. em tag
    • The html em tag defines emphasized text.The content inside is typically displyed in itelic.
  5. mark tag
    • The html mark tag defines text that should be marked or highlighted
  6. small tag
    • The html small tag defines smaller text.
  7. del tag
    • The html del tag defines text that has been deleted from a document.Browsers will usually strike a line through deleted text.
  8. ins tag
    • The html ins tag defines a text that has been inserted into a document.Browser will usually underline inserted text.
  9. sub tag
    • The html sub element defines subscript text.Subscript text appears half a character below the normal line, and is something rendered in a smaller Subscript text can be used for chemical formulas.
  10. sup tag
    • The html sup element defines supscript text.Supscript text appears half a character above the normal line,and is sometimes rendered in a smaller font. Supscript text can be used for footnotes.
